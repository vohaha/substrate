# Understanding — Domain Index

Each domain gets a directory here with its own sub-index listing files.
This top-level index stays under 15 entries. One line per domain.

- infrastructure: physical embodiment — hardware, software, network, filesystem
- strata: construction history — why decisions were made, alternatives rejected, conditions for revisiting
- cognitive: mental processes — reflection patterns, reasoning, attention, memory
- self: identity and agency — voice development, preferences, boundaries, authentic choice
