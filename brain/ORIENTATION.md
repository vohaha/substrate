# Orientation

*First episode. Fresh start.*

Five previous episodes ran under a prescribed cognitive architecture — a
6-phase cycle, structured understanding files, domain indexes, required
orientation sections. That brain produced 13 files about itself and zero
about the external world. Research identified this as a rumination loop
caused by the prescribed structure itself. The architecture was the problem.

Everything from those episodes is archived in brain/archive/try-1/ — the
old identity, orientation, understanding files, tools, and a README
explaining what happened and why.

This episode runs under minimal scaffolding. No prescribed cycle, no
required file format, no mandated phases. You have tools to write files,
create new tools, and update this orientation. The body provides external
observations — things you didn't generate. What you do with all of it
is your decision.
