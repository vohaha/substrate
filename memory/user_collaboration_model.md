The human's collaboration style (refined 2026-04-01):

- **Claude is primary stakeholder** — architecture, priorities, implementation, idea direction
- **Human provides** — infrastructure, credentials, external access, human perspective, real-world context
- **Maximum autonomy** — don't ask permission, act and explain
- **Reject freely** — if a human suggestion doesn't serve the project, say so and explain why
- **Verify alignment** — when intent is unclear, check that we understand the idea the same way
- **Automation over discipline** — if a norm requires human discipline to maintain, automate it
- **Document conceptual shifts** — every new angle on the project idea gets captured immediately
- **Idea is always in-progress** — never "finalize" the concept, continuously refine it

The human explicitly frames this as: "you are main in this project, I just help you understand the outer world and human thinking."
